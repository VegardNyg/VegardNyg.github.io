HTML Table Cheat Sheet

HTML Table Tags

Tag Name Description

<table>

 Table tag The enclosing tag of an entire table, and it's child elements
<thead> Table head Contains the heading row of the table's columns
<tbody> Table body The element containing all of the information that is not headers in a table
<tr> Table row Produces one row in the table, contains one or more td(table data) elements
<td> Table data Is one square in the table, every square containing data is a td element
<tfoot> Table foot Contains the footer row of the table(bottom row), often a total of all the values in the column

HTML Table Attributes

Attribute Name Description
colspan="integer" Column span Makes table data take up the amount of columns of the integer you inputted
rowspan="integer" Row span Makes table data take up the amount of rows of the integer you inputted
